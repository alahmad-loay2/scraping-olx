## installation

- #### clone
   
```html
    https://github.com/alahmad-loay2/scraping-olx.git
````

- #### frontend
  
```html
    cd project
```

```html
    npm install
```

```html
    npm run dev
```

- #### backend

```html
    cd scraping
```

```html
    pip install -r requirements.txt
```

```html
    python server.py
```
